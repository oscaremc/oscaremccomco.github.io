Diseño/Prototipo
Formulario
Esta creado para facilitar la forma en ingresar datos a la base de datos y tambien para retornar los datos en caso de actualizarlos. Este formulario fue creado de la necesidad de simplificar los procesos en el caso comparativo de la interfaz de postman.com donde se agregaban los datos por medio de codigo y se complicaba el acceso todo usuario.
Resultado CRUD
Este espacio fue creado para previsualizar los datos que se agregaron al formulario y confirmar que todo este correctamete ingresado. tambien pensando en la experiencia de usurio y evitar los errores, en comparancion a la experiencia de postman.com donde no se puede previsualizar la informacion agregada a la base de datos. ademas se argrega el boton de eliminar si los datos ingresados nos son los esperado.
Importante
Este proyecto es creado en REACT.JS mostrando un CRUD funcional, intuitivo y con estilos. pero dentro del dominio que se usa en GITHUB PAGE no es posible correrlo y ver su funcionamiento, por eso puedes encontrar el codigo dentro de mis repositorios con nombre "oscaremccomco/SodimacEntrega", o en el siguiente link: https://github.com/oscaremccomco/SodimacEntrega. recuerda que si necesitas una guia facil y practica puedes seguir las insrtucciones que estan en README.md archivo alojado dentro del mismo proyecto.
